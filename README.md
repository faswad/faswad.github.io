# faswad.github.io

الموقع الشخصي للدكتور فراس اسود العبادي، ويتضمن الصفحة الرسمية لتطبيق **Salat_FM 2.6.5** داخل مجلد `salat/`.

## النشر

ارفع محتويات هذا المجلد إلى جذر مستودع GitHub Pages. ملف التطبيق المعتمد موجود في:

`salat/assets/downloads/firas-prayer-display.apk`

يجب الحفاظ على اسم الملف نفسه حتى تبقى روابط التنزيل صالحة.

## تحديث إصدار التطبيق مستقبلاً

1. استبدل ملف APK داخل `salat/assets/downloads/`.
2. حدّث القيم في `salat/assets/js/config.js`.
3. حدّث ملف `firas-prayer-display_SHA256.txt` بالبصمة الجديدة.

عنوان البريد الرسمي لا يظهر كنص في صفحات الموقع. نموذج التواصل يفتح تطبيق البريد لدى الزائر برسالة جاهزة.
