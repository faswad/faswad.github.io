تقرير إعادة تصميم موقع Salat_FM 2.6.5
التاريخ: 19 يوليو 2026

أبرز التغييرات:
- إعادة تصميم صفحة التطبيق بالكامل بمظهر احترافي متناسق مع ألوان التطبيق.
- تقديم التطبيق على أنه مشروع للمساجد أولاً، ثم للهواتف وشاشات Android.
- استخدام الصورتين الجديدتين للوضعين الأفقي والعمودي مع ظهور شريط الأحاديث.
- إنشاء صورة مشاركة اجتماعية جديدة بمقاس 1200 × 630.
- تحسين العرض على الهواتف والشاشات الصغيرة بقائمة متجاوبة وتخطيطات مرنة.
- إضافة عرض منظم لمزايا المسجد، ثم مزايا الهاتف، ثم الاعتمادية والتخصيص.
- إضافة ملاحظة تثبيت احترافية توضح إعادة المحاولة واختلاف إصدارات Android.
- إبقاء عنوان التواصل مخفياً عن الصفحة، مع نموذج يفتح رسالة بريد جاهزة.
- تحديث صفحة الموقع الشخصي بصورة التطبيق الجديدة ووصف يضع المساجد أولاً.
- تحديث شروط الاستخدام وسياسة الخصوصية وتاريخ ملفات sitemap.
- إضافة أداة لتبديل APK وحساب الحجم وبصمة SHA-256 تلقائياً.

الفحوصات:
- فحص روابط الصور والأنماط والبرامج والصفحات المحلية: ناجح.
- فحص تكرار معرفات HTML: ناجح.
- فحص JavaScript بواسطة node --check: ناجح.
- فحص CSS بواسطة tinycss2: ناجح.
- فحص manifest وملفات JSON: ناجح.
- فحص تطابق APK مع ملف SHA-256 وconfig.js: ناجح.
- لا يظهر عنوان البريد الرسمي كنص داخل HTML أو CSS.

تنبيه:
لم يُرفق APK جديد مع طلب تحديث الموقع، لذلك بقي APK الموجود في الموقع القديم.
راجع UPDATE_APK_AR.txt قبل الرفع النهائي إذا كانت لديك نسخة APK أحدث.
