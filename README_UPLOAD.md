# Professional Firas Salat website

Designed for:

- https://faswad.github.io/salat/
- Repository: faswad/faswad.github.io
- APK: https://github.com/faswad/faswad.github.io/releases/latest/download/firas-prayer-display.apk

## Replace the website

1. Extract `salat_professional_replacement.zip`.
2. Open GitHub Desktop.
3. Clone `faswad/faswad.github.io`.
4. Delete the old local `salat` folder.
5. Copy the new extracted `salat` folder into the repository.
6. Commit: `Redesign Salat website professionally`
7. Click `Push origin`.
8. Wait for GitHub Pages deployment.
9. Open https://faswad.github.io/salat/
10. Use a hard refresh if the old design appears.

The APK download address is present directly in HTML and in `assets/js/config.js`.
